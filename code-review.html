Video Description

This video reviews my original Password Manager project,a simple application that stores account information and passwords in a JSON file. I also discuss areas where the code could be improved, such as organization, security, and overall design. Finally, I explain the enhancements I planned to make for the updated version of the project and how those changes connect to the goals of the capstone course.
