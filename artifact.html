SecureVault — Multi‑User Password Manager

A secure, multi-user password manager built with Python, SQLite, and AES encryption.

SecureVault is a fully functional password manager designed with a strong emphasis on security, modularity, and clean software engineering practices. It supports multiple user accounts, encrypts all stored credentials using AES, and provides a simple, user-friendly interface for managing sensitive information.

Key Features

  • Multi-user support: Each user has isolated, encrypted vault data.
  • AES encryption: All passwords and sensitive fields are encrypted using industry‑standard AES.
  • SQLite database: Secure, structured, local data storage with clean schema design.
  • Password generator: Built‑in strong password generation for improved security hygiene.
  • Modular architecture: Organized Python modules for encryption, authentication, UI, and database operations.
  • Secure authentication: Salted hashing, input validation, and safe credential handling.
  • User-friendly interface: Simple, intuitive layout built with Tkinter.
